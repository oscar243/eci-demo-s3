5. Modelos de Precios (Pago por Uso)

S3 utiliza un modelo de precios de pago por uso, cobrando solo por los recursos que realmente utilizas, sin costos iniciales ni compromisos a largo plazo.

5.1 Desglose de Costos de S3

Resumen de Costos de Amazon S3
Almacenamiento Se cobra por GB-mes, varía según clase y región.
Solicitudes y Recuperación de Datos Costos por PUT, GET, COPY, LIST y recuperación.
Transferencia de Datos Se cobra al mover datos fuera de S3.
Gestión y Análisis Inventory, Class Analysis, Tagging, Metadata.
Replicación Costos asociados a la copia entre buckets.
Transformar y Consultar S3 Object Lambda, S3 Select.
AWS Free Tier 5 GB Standard, solicitudes gratis, 100 GB transferencia/mes.

5.2 Clases de Almacenamiento

Amazon S3 ofrece diferentes clases de almacenamiento optimizadas para distintos patrones de acceso y necesidades de costo, desde datos de acceso frecuente hasta archivado a largo plazo.

Clase de Almacenamiento Propósito y Uso Costo y Tarifas Duración Mínima
S3 Standard Acceso frecuente. Ideal para sitios web, aplicaciones y análisis de datos. Alto por GB. Tarifa de solicitudes. No aplica
S3 Intelligent-Tiering Acceso desconocido o cambiante. Optimización automática. Tarifa de monitoreo por objeto. Varía entre costos de acceso frecuente e infrecuente. No aplica
S3 Standard-IA Acceso infrecuente. Para copias de seguridad a largo plazo que requieren recuperación rápida. Bajo por GB. Tarifa de recuperación. 30 días
S3 One Zone-IA Acceso infrecuente, no crítico. Datos que pueden recrearse fácilmente. 20% más barato que Standard-IA. Tarifa de recuperación. 30 días
S3 Glacier Instant Retrieval Archivo con recuperación instantánea. Datos que se acceden una vez al año. Muy bajo por GB. Tarifa de recuperación más alta. 90 días
S3 Glacier Flexible Retrieval Archivo con recuperación flexible. Para copias de seguridad y recuperación de desastres. Muy bajo por GB. La recuperación puede tardar de minutos a horas. 90 días
S3 Glacier Deep Archive Archivo a muy largo plazo. Para retención por 7-10+ años. El más bajo por GB. Recuperación en 12 horas. 180 días
S3 Express One Zone Alto rendimiento y baja latencia. Para aplicaciones sensibles a la latencia. Costo por almacenamiento y cómputo. No aplica

5.3 Solicitudes y Recuperación de Datos

Tipo de Solicitud Descripción Costo Asociado
Solicitudes de Escritura PUT, COPY, POST, LIST. Generalmente más costosas.
Solicitudes de Lectura GET, SELECT. Más económicas que las de escritura.
Solicitudes de Eliminación DELETE, CANCEL. Gratuitas.

5.4 Transferencia de Datos

Tipo de Transferencia Descripción Costo Asociado
Entrada a S3 Mover datos desde Internet hacia un bucket de S3. Gratuita.
Entre buckets en misma región Mover datos entre dos buckets dentro de la misma región de AWS. Gratuita.
A otros servicios AWS en misma región Mover datos de S3 a otro servicio de AWS (ej. EC2, Lambda) en la misma región. Gratuita.
A CloudFront Mover datos de S3 a la red CDN de CloudFront. Gratuita.
Saliente a Internet Mover datos de S3 a un usuario o servicio fuera de la red de AWS. Costo por GB que disminuye a mayor volumen.
Entre diferentes regiones Mover datos de un bucket en una región a otro bucket en una región diferente. Costo por GB.
Transferencia Acelerada Usar servicios como S3 Transfer Acceleration o Puntos de Acceso de Múltiples Regiones para acelerar la transferencia de datos. Costo adicional por GB procesado.

5.5 Gestión y Perspectivas

Función Costo y Propósito
S3 Inventory Genera informes sobre tus objetos para gestión. El costo se basa en la cantidad de objetos listados. Es útil para auditar y planificar.
S3 Storage Class Analysis Analiza automáticamente los patrones de acceso a tus datos para ayudarte a determinar la clase de almacenamiento más adecuada. Se te cobra por el análisis de cada objeto.
S3 Storage Lens Ofrece una vista centralizada de todo tu almacenamiento en S3, con métricas y análisis de uso para ayudarte a optimizar costos y proteger datos. Su costo es por la cantidad de datos monitoreados.
S3 Object Tagging Te permite añadir etiquetas a tus objetos para organizarlos, lo que ayuda en la gestión y automatización. No tiene costo.
S3 Metadata Proporciona metadatos consultables en tiempo casi real para organizar y acelerar la búsqueda de datos. Se te cobra por la cantidad de actualizaciones de objetos.

5.6 Replicación

Tipo de Replicación Cargos Principales
CRR & SRR Almacenamiento en destino, solicitudes PUT, cargos de recuperación y de transferencia saliente (CRR).
Replicación por Lotes Mismos cargos que replicación estándar, más cargos por operaciones por lotes.

5.7 Transformación y Consulta

Función Costo Asociado
S3 Object Lambda Solicitudes GET de S3, cargos de cómputo de Lambda, datos devueltos.
S3 Select y S3 Glacier Select Cantidad de datos escaneados y devueltos.

5.8 Tablas S3 y Nivel Gratuito

Concepto Descripción/Costo
S3 Tables Cargos por almacenamiento, solicitudes, monitoreo de objetos y compactación.
Nivel Gratuito de AWS Incluye 5 GB de S3 Standard, 20,000 GETs, 2,000 PUTs y 100 GB de transferencia saliente por mes.
